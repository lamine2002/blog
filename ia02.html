Classement des langages informatiques : Python consolide sa 1ère place, Lua de retour dans le top 20


Ce mois-ci, Lua s’empare de la 20ème position du classement des langages les plus populaires. Parallèlement, Python, C et Java restent sur le podium.


Posté Le 19 mars 2022 | #IA | Par Lamine NIANG

Lua entre dans le top 20 des langages de programmation


Alors qu’il avait disparu des premières places du classement après 2011, le langage de script Lua fait son grand retour, à la 20ème position des langages de programmation les plus populaires. Celui-ci a gagné 18 places, entre février et mars 2022.


À son apogée en 2011, Lua a brièvement atteint une position dans le top 10. On ne sait pas si cela va se reproduire. Mais il est clair que Lua est en train de rattraper son retard sur le marché du développement de jeux : facile à apprendre, rapide à exécuter, et simple à interfacer avec C. Cela fait de Lua un candidat parfait pour ce travail, a expliqué Paul Jansen, PDG de TIOBE Software.


Cette soudaine popularité peut également s’expliquer par le fait que Roblox, un des jeux les plus tendances du moment, impose aux développeurs de créer des expériences de jeu en utilisant Lua.


Un top 3 qui reste solide


Ce mois-ci, nous retrouvons un podium identique aux classements précédents. Après avoir atteint la première position au mois d’octobre dernier, Python garde le rythme, et signe même une croissance d’utilisation de 3,95 %. C se classe en deuxième position (-2,27 % d’utilisation), et Java ferme la marche (+ 0,74 % d’utilisation).


Découvrez le top 10 des langages de programmation selon l’index TIOBE :


  • Python
  • C
  • Java
  • C++
  • C#
  • JavaScript
  • Visual Basic
  • Assembly language
  • PHP
  • SQL


Quels sont les critères du classement TIOBE ?


Le classement TIOBE est un index de référence, qui, chaque mois, indique la popularité des langages de programmation. Celui-ci s’appuie sur les requêtes des 100 langages de programmation les plus recherchées sur les principaux moteurs de recherche (Google, Bing, Yahoo, Wikipédia, Amazon, YouTube et Baidu). De ce fait, l’objectif du classement n’est pas d’élire le meilleur langage de programmation, mais d’identifier les tendances dans le domaine du développement.
